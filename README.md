# CodeAlpha Tasks

This repository contains all my internship tasks completed for CodeAlpha.

## Tasks

### Task 1 - To-Do List

A simple task manager to add, complete, and delete daily tasks.

### Task 2 - Music Player

A responsive music player with play, pause, next, and previous controls.

### Task 3 - TaskFlow Pro (Project Management Tool)

A collaborative project management tool inspired by Trello / Asana.

### Features:

* User Register / Login
* Create Projects
* Add Tasks
* Assign Tasks to Users
* Add Comments
* Task Boards:

  * To Do
  * In Progress
  * Done
* Frontend + Backend
* Real-time updates using Socket.IO

## Technologies Used

* HTML
* CSS
* JavaScript
* Node.js
* Express.js
* Socket.IO
